Methodology & Architecture

We have used Octakit to help us scrape through thousands of repositories gathering project source code. Octakit is GitHub’s official API toolkit chain that helps users access repositories, GitHub’s insights into user data, contributions, and difficulty of the codebase analyzed with Claude 4.5 Pro.

We have written an automation using Google Opal, where the automation drives the GitHub’s throughput is delivered to the Google Gemini 3 Pro engine driven through an open source model called Gemma 3, which drives the entire code analysis tool which helps us export the entire dataset scraping through over 11,000 entries so far, which are further scrutinized through human intervention.

The Gemma model has been trained to identify the key difference in timestamps across commits, and using identification of file modification metadata provided by the Metadata API we achieve a comprehensive set of metrics. This combined with the users LinkedIn profile and social profiles help us understand the skills of the user, paired with this Metadata API unlocks many key factors such as coding time, efficiency, performance, aptitude, contribution, collaborativeness and metrics about the project such as delivery time, agile score, productivity, KPIs used and deliverables of the project.
